# project style

## blender

* one BU = 1 meter

### fbx export settings

* - [x] Selected Objects
* Scale: 1.0
* Forward: Z Forward
* Up: Y Up
* - [x] !Experimental! Apply Transform

### vertex color shading

* name the material "VERTEX" - will have a generic vertex color shader set up
